Step Analysis PowerBI

Step Analysis

This project is based on the results of my steps every day. I analyze the progress of my steps every day. The tools used are Microsoft Excel and PowerBI. Using Advanced Calculations in DAX.

1. Calculate the Average of steps = AVERAGEX(Activity, [Total Steps])

2. Running percentage increase = DIVIDE([Steps (Running)], [Total Steps])

3. Steps Per Running = CALCULATE([Total Steps], Detail[Activity_Name] = "Running")

4. Steps Per Waliking = CALCULATE([Total Steps], Detail[Activity_Name] = "Walking")

5. Total Steps (Cummulative) = CALCULATE([Total Steps], FILTER(ALLSELECTED('Date'), 'Date'[Date] <= MAX(Activity[Date])))

6. Week Over Week % Change Steps = VAR CurrentWeek = CALCULATE([Total Steps], FILTER(ALL('Date'),'Date'[Week of Year] = SELECTEDVALUE('Date'[Week of Year]))) VAR PreviousWeek = CALCULATE([Total Steps],FILTER(ALL('Date'),'Date'[Week of Year] = SELECTEDVALUE('Date'[Week of Year]) - 1)) RETURN DIVIDE(CurrentWeek - PreviousWeek, PreviousWeek)
